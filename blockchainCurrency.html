Blockchain Currency

The way I came up with this is a bit embarrassing, I asked chat-gpt for a project idea, and it gave me a blockchain and I thought it would be pretty cool to remake a cryptocurrency like Bitcoin, so, that's what I did.

How I started

To start this project I first began by researching how a blockchain actually worked, I had a basic idea however, I didn't really know what I was doing or how I would implement it. So, after researching I made a simple class that would handle all the server and local side things that my currency would use.

Creating the blockchain

In order to create the blockchain what I first had to do was create a list of objects that would contain data about the block and the block before it. After I created a POW (Proof-Of-Work) function that would require work to be done (I used SHA-256) in order to find a valid nonce for that block. After it was found It would be sent to the server and broadcast to the rest of the miners that the block had been found and that they could move on to the next one.

With mining complete and creating and updating blocks being completed I now had to work on transactions and wallets, This was decently simple thanks to a library that would allow me to implement A NIST384p cryptographic signing method, I don't really know the inner workings of the function as of now, however, I am hoping to recreate a function like that in order to understand cryptology on a deeper level. However, with the function a user of the blockchain would get two keys, a private key and a public one, the private one would be used to sign data and confirm that the person who owns the account made the transaction. And the public key is used for other people to verify that the transaction happened. Put them all together, and you now have a working system, but, no GUI.

Wallet GUI

Using custom Tkinter I created what I think is a nice GUI that would allow users to see how much of the currency they had, their public wallet ID, and a button to send currency to another user. This was mostly just plug and play due to how I built my main class so, it was decently simple.
