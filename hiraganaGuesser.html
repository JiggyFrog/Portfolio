Hiragana Guesser

Hiragana is one of the three Japanese alphabets that they use, I made a program that is able to detect hand drawn characters. This was slightly more difficult than an english one due to the characters similarity and complexity

How I started

In order to start this project what I first did was create a simple Pygame application where the user could draw characters that could be outputted in a format that an AI could read. This was somewhat simple as it was really just a drawing application which I had made before so it wasn't anything new to me.

Building the Neural Network

I created the neural network using tensorflow as I have used it before so it wouldn't be much of a challenge, I started by creating an input layer based on the size of my training dataset (28 pixels by 28 pixels). This was then followed by A few hidden layers which are what actually allows the AI to think. I then gave it an output layer that had 49 nodes (one for each character) and told the AI to output a percentage zero to one which would be the probability of the drawn image being that character. This would in turn allow me to guess which character the user had drawn. To glue it all together I would import the model using Keras and then every 120 frames (approx. 2 seconds) it would tell you its prediction in the top right.
