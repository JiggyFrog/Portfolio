Weather App

When I first started to learn how to use APIs I made a weather app. Since it has been 4ish years since I made it and the APIs I used are probably obsolete, I thought it was time for a remake

How I started

Since this project wasn't going to be anything crazy I decided that I was just going to use customTkinter and a weather API (I chose open-meteo because it's free) the first step was converting a city name into coordinates on the globe, for this I used geopy because it was a quick way to convert a city name into coordinates which is what open-meteo uses

Designing the application

I used customTkinter because it is a very easy way to create nice looking GUIs in python. After I had initialized most of the components all I had to do was place them onto the window, this project wasn't very hard or show off many skills, however, I wanted to remake one of my first very old projects just for the fun of it.
