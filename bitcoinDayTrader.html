Bitcoin Day Trader

I wanted to learn how good an AI would be at day trading due to the amount of get rich quick ethereum trading bot ads I was seeing

How I started

I first began by creating a dataset managing class that would manage all the data for bitcoin prices every 15m since 2018 This class was very simple all it did was give me easy access to this data and a really convenient way to plug it into my AI model. I formatted the data so that the AI would be classifying using binary crossentropy to predict that based on the previous six candlesticks if the next candlestick would go up, or down Next I started to work on a class for the model itself which was just using tensorflow and keras to create a simple model

Training

After that I trained the model overnight (~500 epochs) and it was set to go trade, however there was one very important step to making it work I needed to figure out an API to gather new information, and submit trades, for this I chose xeggex.com as they had a very in depth api and seemed really simple to use. One hurdle that I overcame was that the API required authentication to make sure it was me making the trade, which makes sense. However, it required an HMAC Sha-256 Signature. So, I used the hashlib, and hmac libraries in order to create this signature so that it could verify my trades and actually put them in. I traded it for a little until I realized that because bitcoin is not very volatile I was getting destroyed by trading fees, however, if there were no trading fees the bot wouldn't have actually done that bad.


Results

The results were actually pretty good as it had around an 83% training accuracy and a 72% field accuracy. I am quite proud of this bot because it worked really well for what it is I'm thinking about upgrading this project later but training it on a more volatile coin so that maybe it can turn a profit.
